# PotPlayer 皮肤

这是一套为PotPlayer播放器设计的自定义皮肤，包含音频和视频播放界面的完整皮肤组件。

## 功能特点

- 现代化UI设计，简洁美观
- 包含音频迷你控制器、音频皮肤和视频皮肤
- 自定义控件和图标
- 响应式布局，适配不同窗口大小

## 文件结构

- `AudioMini.xml`: 音频迷你控制器界面
- `AudioSkin.xml`: 音频播放界面皮肤
- `VideoSkin.xml`: 视频播放界面皮肤
- `common.xml`: 通用样式和控件定义
- `module.xml`: 模块配置
- `string.xml`: 字符串资源
- `logo.png` 和 `main.png`: 皮肤图片资源

## 自定义

您可以通过修改相应的XML文件来自定义皮肤外观：

- 修改颜色和字体：编辑 `common.xml`
- 调整控件布局：编辑对应的界面XML文件
- 替换图片资源：替换 `logo.png` 和 `main.png`

- 确保您的PotPlayer版本支持自定义皮肤

- 皮肤可能与某些PotPlayer版本不兼容
